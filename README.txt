Fly Pantie 〜浮気の証拠を隠滅せよ〜
Ver 1.02

起動方法:お使いのOSに合わせたフォルダを開き、それぞれのOSに合わせた実行方法で実行してください。

あらすじ:彼女がいながらも浮気をしてしまってる"君"は、彼女が部屋に帰ってきたときに浮気相手のパンティーが手元にあることに気が付く！
	 "君"は彼女にみられる前にそれを部屋の外へ投げ捨ててしまわなくてはならない！！

内容	:浮気相手のパンティーを彼女に見つからないように窓の外へ投げ捨てます。プレイヤーは飛んでいるパンティーを操作して窓の外へ飛ばします。
	 ハンガーに接触するか、床に落ちるとゲームオーバー、ステージ右端の窓に接触するとゲームクリアです。
	 それぞれ性能の違う5つのパンティーでクリアを目指せ！

操作方法:ゲーム画面ではSpaceキーを押している間パンティーが上昇します。パンティーは自動で画面の右へ飛んでいきます。
	 タイトル、パンツセレクト、ゲームオーバー、ゲームクリア画面ではクリックで各項目を選択します。
	 彼女に見つかる前にパンティーを部屋の外に投げ捨てなくてはならないのでゲームは"START"を選択するとすぐに始まります。
	 また、タイトル画面、パンツセレクト画面ではSpaceキーでSTARTボタンを、ゲームオーバー、ゲームクリア画面では、
	 "もう一度"をクリックしたときと同じ動作をします。
	 いずれの画面でもESCキーでゲーム終了になります。

注釈	:画面サイズは16:9を想定しています。起動画面でお使いのデバイスに合わせた16:9の解像度をお選びください。
	 "ふんどし"は開発者でさえクリアできない難易度です。ご注意ください。
